For Senior Encounter, a two week internship/shadowing experience with a professional in a chosen field, Us: Alex Boulay and Kaikai Du were tasked with utilizing skills of prompt engineering to have ChatGPT an OpenAI chat bot, write code that would allow us to input Chest X-Ray images and output a prediction on whether the patient has or does not have pneumonia.

To accomplish this we dived into the introduction of prompt engineering, a style of engineering that utilizes A.I chatbots and sends specific prompts to accomplish a task. We used this by splitting our goal into smaller steps that we could then use to send specific prompts to ChatGPT and would allow for the code we received to be accurate. We determined that ChatGPT is incredibly effective at writing code when prompt engineering is applied to most efficiently use the chatbot.

Technical Details:

While most of the code that we used to accomplish this was written by ChatGPT there were still technical requirements that we needed to do before it would work on our own devices. We needed a fresh install of the newest version of Python 3, 3.11, as well as an install of PyTorch an open source framework for deep learning and neural networks that uses python's engine. Documentation for the install of PyTorch can be found on their website Pytorch Documentation, The installation only requires a pip3 install or through other means listed.

A second challenge that we faced was the importing of the dataset that we used to train our model. We gathered our dataset from the MedMNIST Dataset, this dataset is open source and features both 2D and 3D datasets. We chose Pneumonia as our main dataset because of its straightforward nature as well as its Binary Classification which would greatly simplify how we trained our model. While MedMNIST structures their datasets into a NumPy array format with the .npz file format, it comes with documentation that shows examples on how to use built-in PyTorch integration to use the MedMNIST dataset for neural networking. The MedMNIST dataset can be found on their website, MedMNIST. As well as their github which provides a guide for implementing MedMNIST dataset into PyTorch, MedMNIST Github.
