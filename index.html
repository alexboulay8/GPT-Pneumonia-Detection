Getting started

Project Summary:

This is a project done by two high school seniors at the Johns Hopkins Medicine Technology Innovation Center with no previous knowledge of machine learning as part of their two week internship as part of their schools "encounter" program. The project focused on testing ChatGPT's coding capabilities and as such the code is largely written by AI. The resulting neural network is able to distinguish between chest x-rays of lungs with and without Pneumonia at an 85% accuracy.
